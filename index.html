Raft Algorithm - Scala

source

Written for West London Hack Night (January 2015).

This is an implementation of a distributed consensus algorithm in Scala and Akka.

It was a team effort, and one of the most difficult hacks we've attempted. We got most of it done in the allotted 90 minutes, but there were a lot of bugs that needed fixing, so this is a post-hack fork with the basics working.

Tube Stations Algorithm - Python

source

Algorithm to compute the minimum number of London tube station names required to cover all letters of the alphabet.

I used to ask this to junior developers in job interviews. This was a typical whiteboard programming exercise, and the company subscribed to the "make the candidate as uncomfortable as possible" interviewing style, so candidates were expected to ramble on about heuristics for half on hour rather than actually answer the question as asked.

I wanted to find a correct solution, and after reading up on dynamic programming algorithms I came up with this.
